# Helpdesk Technician Dashboard

Windows helpdesk dashboard written in Python using Tkinter and psutil.

## Features

### Live Status
- CPU usage
- RAM usage
- Disk C usage
- Host name
- IP address

### Diagnostics
- System info
- Network info
- Ping test (8.8.8.8)

### Fix & Repair
- Flush DNS
- Clear temp files
- Reset network stack
- Restart explorer
- Restart print spooler
- Run SFC
- Run DISM

### Tools
- Task Manager
- Services
- Event Viewer
- Network Connections
- Control Panel

## Requirements
- Windows 10/11
- Python 3
- `pip install psutil`
